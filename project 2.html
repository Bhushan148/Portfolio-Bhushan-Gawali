Project Details

Bank Loan Analysis Project

Tools : Power BI (Interactive dashboards and visualizations), SQL (Data extraction and manipulation)

Repository :
Check out the project on GitHub and access the files here : Bank Loan Analysis Project

Project Objective :
The goal of this project is to analyze bank loan data to gain insights into lending activities, identify key performance indicators (KPIs), and track loan performance. This analysis will help in making data-driven decisions and understanding the financial health of the bank's lending portfolio.

Description :
This project involves importing data into SQL, running queries to extract and manipulate the data, and then visualizing the results in Power BI dashboards. The project is divided into three main dashboards:


Project Steps :
  1. Data Import and Preparation:
    • Imported loan data into SQL.
    • Wrote SQL queries to clean and manipulate the data to solve business problems.
    • Connected Power BI to the SQL database and imported the cleaned data.
    • Further cleaned and transformed the data in Power BI as needed.
  2. Dashboard 1 : Summary
    • Key Metrics:
      • Total Loan Applications: Number of loan applications, including Month-to-Date (MTD) and Month-over-Month (MoM) changes.
      • Total Funded Amount: Total funds disbursed, with MTD and MoM analysis.
      • Total Amount Received: Total repayments, with MTD and MoM analysis.
      • Average Interest Rate: Average rate across loans, with MTD and MoM changes.
      • Average Debt-to-Income Ratio (DTI): Borrower financial health, with MTD and MoM tracking.
    • Good Loan vs. Bad Loan KPIs:
      • Good Loan: Application percentage, number of applications, funded amount, and total received amount.
      • Bad Loan: Application percentage, number of applications, funded amount, and total received amount.
    • Loan Status Grid View: Overview by 'Loan Status' showing key metrics.
  3. Dashboard 2 : Overview
    • Key Metrics :
      • Monthly Trends by Issue Date: Seasonal and long-term lending trends.
      • Regional Analysis by State: Lending activity and regional disparities.
      • Loan Term Analysis: Distribution of loan terms.
      • Employee Length Analysis: Lending metrics by employment length.
      • Loan Purpose Breakdown: Metrics based on loan purposes.
      • Home Ownership Analysis: Impact of home ownership on loans.
  4. Dashboard 3 : Details
    • Objective: Comprehensive view of key loan metrics and borrower profiles
Tools Used :
