Data Entry Form Project

Tools : Microsoft Excel (Power Query, Excel Formulas, Macros,Sheet Protection)


Repository:
Check out the project and access the files here : Data Entry Form - Excel

Project Objective :
To create a simple yet effective registration form for meetings using Microsoft Excel, featuring dependent dropdown lists, functional macros, and protected sheets with specific unlocked cells for user interaction.


Steps and Challenges :

Creating the Registration Form:
  • Designed a basic registration form to collect user information for meeting purposes.
Adding Dependent Dropdown Lists :
  • Problem: Needed to include dependent dropdown lists for State and City.
  • Solution:
    • Used Power Query to call data from the web and retrieve all state and city data.
    • Appended all the data into a single table.
    • Cleaned the data to ensure accuracy and usability.
    • Applied Excel filter formula to link each city to its respective state.
Using Excel Filter Formula :
  • Problem: Ensuring that only cities corresponding to the selected state are shown.
  • Solution:Utilized the filter formula in Excel to separate all cities for a particular state.
Incorporating Macros :
  • Problem: Needed to add functionality for "Clear" and "Submit" buttons.
  • Solution:
    • Developed macros for the buttons to perform desired actions.
    • Ensured that data entered in the form is stored in another sheet upon submission.
Protecting the Sheet :
  • Problem: Required certain cells to be protected while others remained editable, including buttons and checkboxes.
  • Solution:
    • Protected the sheet while linking specific cells to the developer button.
    • Ensured these linked cells were unlocked to maintain functionality.
Finalizing the Form :
  • Created a fully interactive and user-friendly registration form that meets all requirements.
Outcome and Results :
  • Successfully developed a dynamic registration form with dependent dropdown lists.
  • Implemented functional macros to enhance user experience and data handling.
  • Achieved a balance between protecting the sheet and allowing necessary user interactions.
Topics Covered in the Project :
  1. Excel Power Query :
    • Data retrieval from web sources.
    • Data cleaning and transformation.
  2. Excel Formulas :
    • Filter formulas for dependent dropdown lists.
  3. Macros:
    • Writing and implementing macros for button functionalities.
    • Handling data submission and clearing form entries.
  4. Sheet Protection :
    • Techniques to protect and unprotect specific cells.
    • Linking cells to developer controls for interactive elements.
  5. User Experience :
    • Designing an intuitive and user-friendly interface.
    • Ensuring smooth functionality and data integrity.


Conclusion

This project demonstrates the effective use of Excel for creating interactive forms, handling data dynamically, and using advanced features like Power Query and macros. The end result is a robust registration form that can be used for various meeting purposes, ensuring ease of use and reliability.
