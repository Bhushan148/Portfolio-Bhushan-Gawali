Web Scraping with Save HTML Webpage

Objective :
In this project, I aimed to scrape data by first saving the complete webpage locally. This method allowed me to extract information step by step and ensure the accuracy of the extracted data.

Technologies Used :
  • Python
  • Requests library
  • BeautifulSoup library
  • Pandas
  • Complete Webpage Here
Description :
In this project, I :
  1. Saved the entire webpage using the Requests library to ensure I had a complete and static version of the page content.
  2. Extracted various pieces of information from the saved HTML content step by step using BeautifulSoup.
  3. Combined all the extracted data into a comprehensive codebase that scrapes all necessary information in one go.
  4. Stored the extracted information in a structured table format using Pandas for easy analysis and access.
Outcome :
  • The project successfully demonstrated the process of web scraping by first saving the entire webpage, ensuring data accuracy and completeness, and then extracting and structuring the information into a tabular format.

Web Scraping IMDB Webpage

Objective :
The goal of this project was to perform live web scraping on the IMDB website, extracting movie data and storing it in a table format.

Technologies Used :
  • Python
  • Requests library
  • BeautifulSoup library
  • Pandas
  • https://www.imdb.com/chart/top/
Description :
In this project, I :
  1. Implemented live web scraping techniques to fetch data directly from the IMDB website using the Requests library.
  2. Parsed the HTML content using BeautifulSoup to extract relevant movie information such as titles, ratings, and release dates.
  3. Structured the extracted data into a Pandas DataFrame for easy analysis and storage.
  4. Saved the DataFrame to a file for future use and analysis.
Outcome :
  • The project successfully scraped live data from the IMDB website, extracted and stored the movie information in a tabular format, and demonstrated effective live web scraping techniques.

Web Scraping Wikipedia Tables

Objective :
This project aimed to scrape table data from Wikipedia, specifically focusing on the 8th table on a given page, which contained population data.

Technologies Used :
  • Python
  • Requests library
  • BeautifulSoup library
  • Pandas
  • https://en.wikipedia.org/wiki/Demographics_of_India
Description :
In this project, I :
  1. Used the Requests library to fetch the HTML content of a Wikipedia page containing multiple tables.
  2. Parsed the HTML content using BeautifulSoup to locate and extract data from the 8th table.
  3. Converted the extracted table data into a Pandas DataFrame for structured storage and analysis.
  4. Performed data cleaning and preprocessing to ensure the extracted data was accurate and ready for further analysis.
Outcome :
  • The project successfully extracted population data from the specified table on a Wikipedia page, demonstrating the ability to handle structured data and extract meaningful information from web tables.
