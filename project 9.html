Project Details

Bank Loan Report Dashboard


Tools : Excel

Repository :
Check out the project and access the files here : Bank Loan Report Dashboard

Project Overview :
This project, titled "Bank Loan Report Dashboard," is inspired by a YouTube tutorial and focuses on creating a fully automated Excel dashboard for bank loan analysis. The dataset used for this project was sourced from the same tutorial, imported, and analyzed to generate key insights into the bank's loan portfolio.

Objective :

The goal of this project is to design a comprehensive dashboard that provides clear and actionable insights into various aspects of the bank's loan operations, including loan applications, funded amounts, repayments, interest rates, and more.

Dashboard 1 : Summary

This dashboard provides a high-level overview of the key metrics related to the bank's loan activities.

Key Metrics:
  • Total Loan Applications:
    • Displays the number of loan applications.
    • Includes Month-to-Date (MTD) and Month-over-Month (MoM) changes to track trends.
  • Total Funded Amount :
    • Shows the total amount of funds disbursed.
    • Analyzes MTD and MoM changes to monitor loan funding trends.
  • Total Amount Received :
    • Displays the total repayments received from borrowers.
    • Includes MTD and MoM analysis to assess repayment patterns.
  • Average Interest Rate :
    • Calculates the average interest rate across all loans.
    • Tracks MTD and MoM changes to understand interest rate trends.
  • Average Debt-to-Income Ratio (DTI) :
    • Measures the average DTI ratio, reflecting borrowers' financial health.
    • Includes MTD and MoM tracking to monitor changes in borrower risk profiles.
  • Good Loan vs. Bad Loan KPIs :
    • Good Loan :
      • Displays the percentage and number of applications categorized as 'Good Loans.'
      • Includes metrics such as the funded amount and total received amount for good loans.
    • Bad Loan :
      • Shows the percentage and number of applications categorized as 'Bad Loans.'
      • Includes metrics such as the funded amount and total received amount for bad loans.
  • Loan Status Grid View :
    • Overview by Loan Status: Provides a grid view that breaks down key metrics by different loan statuses, such as Active, Closed, or Defaulted.
Dashboard 2 : Overview

This dashboard delves deeper into specific trends and analyses across different dimensions of the loan data.

Key Metrics :
  • Monthly Trends by Issue Date :
    • Analyzes loan issuance over time to identify seasonal and long-term lending trends.
  • Regional Analysis by State :
    • Examines lending activity across different states, highlighting regional disparities and performance.
  • Loan Term Analysis :
    • Breaks down the distribution of loans by their terms (e.g., short-term, medium-term, long-term), offering insights into borrower preferences and risk.
  • Employee Length Analysis :
    • Investigates how the length of borrowers' employment affects loan approval, funding, and repayment metrics.
  • Loan Purpose Breakdown :
    • Analyzes metrics based on the stated purposes of loans (e.g., education, home improvement, debt consolidation).
  • Home Ownership Analysis :
    • Assesses the impact of home ownership status on loan application success, interest rates, and repayment performance.
Implementation Details :
  • Data Importation :
    • Import the dataset from your device into Excel.
    • Ensure the dataset is cleaned and formatted correctly, with all necessary columns available for analysis.
  • Data Analysis :
    • Use Excel formulas and PivotTables to calculate KPIs and create visualizations.
    • Automate MTD and MoM calculations using dynamic Excel functions.
  • Dashboard Design :
    • Use Excel's charting tools to create visualizations such as bar charts, line charts, and pie charts.
    • Design the layout to be intuitive and easy to navigate, ensuring that key metrics are prominently displayed.
  • Automation :
    • Set up the dashboard to automatically update when new data is added, using Excel's data connections and refresh options.
    • Ensure all calculations are dynamic, enabling real-time updates as the data changes.
Conclusion :

This project showcases the power of Excel in creating dynamic and informative dashboards for financial analysis. By automating the data analysis and visualization processes, the dashboard provides real-time insights that can help banks make data-driven decisions about their loan portfolios.
