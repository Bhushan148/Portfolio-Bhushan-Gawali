Project Details

Smartphone Market Insights Analysis

Tools : MySQL, Python(Pandas, Matplotlib, Seaborn, MySQL connector)

Repository :
Check out the project on GitHub and access the files here : Smartphone Market Insights Analysis

Project Objective :
Smartphone Market Insights Analysis" project delivers a comprehensive examination of the smartphone market using SQL and Python. The analysis spans various factors such as prices, features, user ratings, and technical specifications of different smartphone models. By leveraging a detailed dataset that includes information on each smartphone's price, battery life, camera quality, processor type, and other relevant attributes, this project aims to provide valuable insights for both consumers and manufacturers.

Description :
This project involves importing data into SQL, running queries to extract and manipulate the data, and then visualizing the results in Power BI dashboards. The project is divided into three main dashboards:


Project Steps :
  1. Data Extraction and SQL Queries :
    • The analysis begins with writing SQL queries in MySQL to extract relevant data from the dataset.
    • Queries focus on different aspects such as price ranges, user ratings, processor types, and more.
  2. Connecting MySQL with Python :
    • Utilize the mysql.connector library to connect the MySQL database with Python.
    • Fetch the query results into Python for further analysis .
  3. Data Manipulation and Analysis with Pandas
    • Use Pandas for fetch date, manipulation, and visualization.
    • Perform various calculations to derive insights from the dataset .
  4. Data Visualization with Matplotlib and Seaborn
    • Create visualizations to represent the insights derived from the analysis .
    • Charts and graphs include distribution plots, bar charts, and scatter plots to visualize trends and comparisons .
Technologies Used :
  • SQL: Used for querying the dataset to extract valuable insights.
  • Python : Employed to connect to the SQL database using mysql.connector and to perform further analysis.
  • Pandas: Utilized for data manipulation and analysis.
  • Matplotlib and Seaborn : Deployed for data visualization
Outcome :
  • This project aims to provide consumers with informed purchasing decisions and help manufacturers understand market trends and consumer preferences. By analyzing the data, valuable insights such as popular price ranges, user satisfaction with different brands and processor types, and trends in display resolution and camera quality are revealed. These insights are visualized through various charts and graphs, making the information easily accessible and interpretable.
